# ixyz-nrm 使用说明

欢迎使用 ixyz-nrm，一个用于管理镜像源的工具。

## 安装

npm install -g ixyz-nrm

## 使用方法

ixyz-tool <command> [options]


## 命令

- `use`: 切换镜像源
- `add`: 添加镜像源
- `del`: 删除镜像源

## 选项

- `-version`, `--version`: 显示版本号
- `-h`, `--help`: 显示帮助信息


## 支持

如果您在使用过程中遇到任何问题，请在 [GitHub](https://github.com/ixyzorg/ixyz-nrm/issues) 上提交 issue。
